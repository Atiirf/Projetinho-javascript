# Inventário Espacial

Projeto de um painel de controle de inventário para nave espacial. Desenvolvido para a disciplina de Desenvolvimento de Aplicações.

## Funcionalidades
- Cadastro de itens com nome, quantidade e classificação crítico
- Listagem de todos os itens cadastrados
- Filtro para exibir apenas itens críticos
- Cálculo e exibição do total de itens
- Destaque visual para itens críticos (cor vermelha)

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Estrutura do projeto
- `index.html` - estrutura da página
- `style.css` - estilização
- `script.js` - lógica do inventário

## Como usar
1. Clone ou baixe o repositório
2. Abra o arquivo `index.html` no navegador
3. Utilize o formulário para cadastrar novos itens
4. Use os botões de rádio para filtrar a lista

## Detalhes técnicos
- Array inicial com 3 itens de exemplo
- Funções em arrow function
- Manipulação de DOM para atualizar a tela
- Métodos map, filter e reduce para processar os dados
- Eventos de submit e change para interação
